CSS 사용 단위 및 값

- 요소의 특징적인(display) 속성을 제외하고, 요소에 직접적으로 스타일이 적용되어 있지 않다면, 부모로부터 상속 받는다.

크기

글자크기 단위
px(절대값)
em(상대값, 배수) : 부모 요소 기준
%(상대값, 백분율) : 부모 요소 기준
rem(상대값, root em) : html 설정값 기준
면적크기 단위(.box크기 200px)
400px
2em(글자크기 단위)
200% (부모 dl의 가로크기 400px의 2배 => 800px)

컬러단위

  • #ffddcc / #FDC : HEX 코드 값, #을 사용, 3~6자리 표기
  • rgb(r,g,b) : rgb(255,0,0)
  • rgba(r,g,b,alpha) : alpha값은 0~1 사이의 숫자로 투명도 조절
  • 관용색명 : 실무에서는 사용하지 않음
  • hsl() : 왠만하면 사용하지 않음
