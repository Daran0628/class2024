1. 글자색 => color

2. 글자 크기 => font-size

3. 글꼴 => font-family

4. 굵은글자 => font-weight

5. 기울기글자 => font-style

6. 행간(행의 높이값) => line-height
- 행과 행 사이 여백은 여백값과 글자크기를 더한 값을 line-height에 적용

7. 자간(글자와 글자사이 간격) => letter-spacing

8. 단어간격(단어와 단어 사이 간격) => word-spacing

9. 글자에 줄긋기 => text-decoration : [underline(밑줄) / line-through(취소선)]

10. 영역에서의 글자 가로 정렬 => text-align

11. 알파벳 대소문자 변환 => text-transform : [uppercase(모두 대문자로) / LOWERCASE(모두 소문자로) / capitalize(첫글자를 대문자로)]

12. 알파벳 소문자를 소문자 크기의 대문자로 변환 => Font-Variant : Small-Caps

13. 단락의 첫글자 들여쓰기 => text-indent

14. 자동 줄바꿈 => white-space / 가로면적보다 길이가 긴 글을 자동으로 줄이 바뀌게 하는 속성
