컬러단위

  • #ffddcc / #FDC : HEX 코드 값, #을 사용 3~6자리
  • rgb(r,g,b) : rgb(0,0,0) black rgb(255,255,255) white
  • rgba(r,g,b,alpha) : alpha 값은 0~1 사이의 숫자로 투명도 조절
  • 관용색명 : 실무에서는 사용하지 않음
  • hsl(hue,saturation,lightness) :왠만하면 사용하지 않음
