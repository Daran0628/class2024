CSS 개론

  • CSS(Cascading Style Sheet)
  • 웹문서의 스타일을 미리 선언해 놓은 문서
  • 웹문서의 폰트, 컬러, 크기 등을 표현, 특히 레이아웃 디자인에 반드시 필요
  • 스타일링 할 대상이 반드시 필요, 그 대상은 태그 요소가 됨
  • 선택자(대상)의 이름이 동일하면 나중에 작성된 것이 적용되고, 이름이 다를 때는 정해진 우선 순위에 따라 적용

선언 방법

내부 선언(Intenal / Embed)

  • 스타일링 하려는 html 문서에 직접 선언
  • 선언된 페이지에만 스타일링
  • head 요소 내부에 style 태그를 이용하여 선언(작성)

외부 선언(External / Linked)

  • 별도의 css 파일을 작성하여 스타일링하려는 문서에 링크(연결)하는 방법
  • 여러 페이지에 동일한 스타일링이 가능
  • head 요소 내부에 link 태그를 이용하여 css 파일을 연결

요소에 직접 선언(Inline)

  • 태그에 style 속성 이용하여 직접 선언
  • 테스트 용도로만 사용, 리액트에서 가끔 사용됨

기초 문법

선택자 {속성: 값; 속성: 값; ...}
selector {property : value; property: value; ...} h1 {color: red;}
