입력(input) 양식(서식, form) 관련 요소

- 사용자로부터 데이터를 입력받기 위한 요소

- forms 요소들은 name과 value 속성이 필수이지만, - DB(DataBase)와 관련된 속성이므로 db와 관련없는 직무라면 생략

- form 태그 속성 : method, action method(전송방식) => get(default), post action(데이터를 처리할 파일 경로)

입력양식 태그 속성

  • placeholder : 사용자가 직접입력하는 인풋요소에 미리보기 문구
  • required : 필수 요소를 나타내는 속성
  • selected : select요소의 option 태그에 사용
  • checked : 체크박스와 라디오 요소에 사용
  • readonly : 사용자 할 수 없는 속성
fieldset 그룹의 heading 예) 추가정보
