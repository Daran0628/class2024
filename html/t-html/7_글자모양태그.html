글자모양태그(Text Formatting)

- 기본 스타일을 가지고 있고, 태그이름이 스타일을 의미하고 있지만, 스타일 때문에 사용하지말고, 의미를 확인하고 사용할 것!!!!!

b 태그 :
- bold체의 의미,
- 의미가 없는 태그
i 태그 :
- italic체 의미
- 전문용어, 다른언어, 화자 의견, 관용어구, etc
u 태그
- underline(밑줄)
- 철자가 틀린 글자를 표시
s 태그
- strike(취소선)
- 더 이상 유효하지 않은 글자를 표시
ins 태그
- insert(삽입), 밑줄
- 내용이 변경되어 새로 삽입된 텍스트
del 태그
- delete(삭제), 취소선
- 내용이 변경되어 삭제해야 하는 텍스트
small 태그
- 부수적인 설명 등, 권장하지 않음
sub 태그 & sup 태그
- sup(윗 첨자) : 2n, O2
- sub(아래 첨자) : H2O
strong 태그
- bold체로 표현됨
- important(중요) text
em 태그
- italic체로 표현됨
- emphasized(강조) text
- 일부 음성브라우저는 큰소리로 읽어주거나 두번 읽어줌
mark 태그
- 형광펜 효과
