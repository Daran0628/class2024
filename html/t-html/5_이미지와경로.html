이미지 태그와 경로의 이해

이미지 태그

이미지 태그의 필수 속성

* src
- source(자료[(자료의) 출처])
* alt
- alternate text(대체 글자)
- 엑박일 때 대신 보여질 텍스트
- 대체 텍스트는 이미지가 전달하는 정보와 동등한 정보를 포함한 텍스트로 작성
- alt 속성 자체를 생략하면 장애인차별금지법에 위배되어 벌금형
- 정보 전달이 목적이 아닌 경우, 속성명만 작성하고 값을 생략

경로의 이해

절대 경로

- http로 시작하는 도메인 주소와 디렉토리 이름, 파일명까지 다 작성

상대 경로

  • 경로를 입력한 파일의 위치에서부터 시작
  • ./

    (현재 디렉토리) : 보여질 문서의 위치[디렉터리]

    생략이 가능 : 일반적으로 html에서는 생략, 리액트 등에서 ./를 사용하는 관습이 있음

  • ../

    (부모 디렉토리) : 보여질 문서가 있는 디렉토리 위치
