글자모양태그(Text Formatting)

- 기본 스타일을 가지고 태그 이름이 스타일을 의미하지만! 스타일 때문이 아닌 목적의 의미에 맞게 사용!!!

b 태그:
- bold체의 의미
- 의미가 없는 태그
i 태그:
- italic체의 의미
- 전문 용어(CSS), 다른 언어, 개념/의견, 관용어구, etc.
u 태그
- underline(밑줄)
- 철자가 틀린 글자를 표시
s 태그
- strike(취소선)
- 더이상 유효하지 않은 글자를 표시
ins 태그
- insert(삽입), 밑줄
- 내용이 변경되어 새로 삽입된 텍스트
del 태그
- deletet(삭제), 취소선
- 내용이 변경되어 삭제해야하는 텍스트
samll 태그
- 저작권
- 부수적인 설명
sub 태그 & sup 태그
- sup(윗 첨자) : 2n
- sub(아래 첨자) : H2O
strong 태그
- bold 체
- important(중요) text
em 태그
- italic 체
- emphasized(강조) text
- 일부 음성브라우저는 큰소리로 읽거나 두 번 읽음
mark 태그
- 형광펜효과
